# FeedMe

Developed for [WildHacks II](https://wildhacksii.devpost.com/)
## Client Deployment


```bash
yarn install
yarn start
```
or
```bash
npm install
npm start
```

## Server Deployment


```bash
pip install -r requirements. txt
python app.py
```

## Socket Deployment


```bash
pip install -r requirements. txt
python socket-server.py
```
